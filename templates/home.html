Fake News Fact Checker



About

The aim of my project is to create a system to detect whether a statement entered by the user is fact or fiction. The website will categorise the text as one of 6 labels:

  • False
  • Half True
  • Mostly True
  • True
  • Barely True
  • Pants On Fire

This is system is only 21% accurate for 6 labels and should not be treated as a complete system.
